# NOFX Skills for OpenClaw

OpenClaw agent skills for the [NOFX](https://github.com/NoFxAiOS/nofx) crypto AI trading platform.

## Skills

### `nofx`
Core NOFX platform skill — API interactions, strategy management, trader control, market data, and AI trading signals.

### `nofx-ai500-report`
Automated AI500 market intelligence reports — monitors coin selections, analyzes OI, institutional fund flows, K-line technicals, and generates periodic trading signal summaries.

## Installation

```bash
# Via ClawHub
clawhub install nofx
clawhub install nofx-ai500-report

# Or manually copy to your OpenClaw skills directory
cp -r nofx ~/.openclaw/workspace/skills/
cp -r nofx-ai500-report ~/.openclaw/workspace/skills/
```
